---

### ❌ 如果看到错误（需要处理）：

```
❌ 错误：npm命令不可用
```
**→ 说明仍在旧的PowerShell会话，需要关闭重开**

或

```
npm ERR! code ETIMEDOUT
npm ERR! network timeout
```
**→ 网络问题，可能需要重试或更换网络**

---

### ⏸️ 如果卡在某处不动

检查是否在等待输入，按回车键继续。

## 快速检查命令

您也可以在命令行窗口输入以下命令快速检查：

```cmd
dir node_modules
```

- 如果显示"找不到文件"：依赖未安装
- 如果显示文件夹列表：依赖已安装 ✅
